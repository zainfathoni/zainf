# Mini Shop — Techrawih x ReactID Demo

Demo app untuk live coding session. Dibangun dengan bantuan AI dari nol dalam 30 menit.

## Struktur Demo

```
Step 1 — Scaffolding (Vite + React)
Step 2 — Data produk (src/data/products.js)
Step 3 — Komponen ProductCard (src/components/ProductCard.jsx)
Step 4 — State + CartSummary (src/App.jsx)
Step 5 — Polish & Styling
```

## Cara Menjalankan

```bash
npm create vite@latest mini-shop -- --template react
cd mini-shop
npm install

# Salin file-file dari demo ini ke src/
# Kemudian:
npm run dev
```
