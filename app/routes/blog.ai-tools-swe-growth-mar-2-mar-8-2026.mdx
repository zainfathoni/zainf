## 📈 Trends

**🚀 Emerging:**
- Copilot CLI sebagai legitimate alternative — bukan lagi 'second class'
- Per-request pricing model (Codex) vs per-token (Opus) — pricing war baru
- Cross-tool OAuth interop (Copilot auth di pi.dev, Opencode, dll)
- Oh My OpenAgent — agent orchestration evolving from tool wrappers
- M-shaped people — AI push PM dan SWE jadi generalist

**📊 Continuing:**
- Subscription fatigue & regional pricing inequality
- Multi-provider strategy — jangan taruh semua telur di satu keranjang
- Agent harness > model choice consensus makin kuat
- Budget combo hunting (Claude + Amp free + Copilot $10)

**📉 Fading:**
- Single-tool loyalty — semua orang mix & match sekarang
- Antigravity/Windsurf hype — makin banyak complaint
- IDE-first approach — CLI tools makin dominan

## 🎯 Key Topics Discussed

### 1. Claude Suspension Wave & Multi-Provider Survival

- Mas Faris dapat email suspension dari Anthropic — investigasi 'suspicious signals'
- Claude down bersamaan dengan AWS data center incident (Middle East)
- Respons komunitas: jangan ketergantungan 1 provider
- Subscription pricing war dimulai — siapa yg bisa combine paling hemat

### 2. Subscription Economics & 'Kere Hore' Status

- Di Brazil: $200 = 60% gaji. Di Indonesia: bisa full gaji
- Regional pricing masih jauh — provider masih cari cuan
- Strategy: combine Claude Max $100 + kantor punya Teams + Amp gratis
- Mas Desilino akhirnya uninstall OpenClaw — budget vs kebutuhan reality check
- Kilo sering kasih 'bansos' free credits — tapi tidak reliable
- 'Sobat mentok kanan' vs 'kaum mendang mending' — class divide di AI tools semakin terasa

### 3. Agent Harness > Model Choice

- Sama-sama Opus, beda harnessing = beda hasil drastis
- System prompt bukan satu-satunya faktor — kapan dipanggil, context apa, kapan cache hit
- Compound engineering di Amp — tiap beads punya field skill dan mode, tinggal handoff
- Claude Code plugin compound-engineering makin populer tapi boros token

### 4. Singapore AI Bansos vs Indonesia Reality

- Singapore: free premium AI subs dari pemerintah, H2 2026
- Indonesia: subs AI bisa setengah sampai full gaji
- Regional pricing discussion — kapan semua adopt?
- Leverage sebagai business expense jadi solusi pragmatis

### 5. Security Concerns: Supply Chain + Fake Clones

- Supply chain attack + prompt injection case study — simple theory, scary execution
- Mas Faris salah install fake OpenClaw clone — nomor 1 di web ternyata palsu
- Pindah ke nanobot + ZeroClaw sebagai alternative
- Perlu double-triple waspada kalau serahkan prod infra ke LLM

### 6. Code Reviews Are Dead & M-Shaped People

- SWEs perlu product thinking, PMs perlu technical depth
- AI push semua orang jadi M-shaped person
- Quality vs YOLO tension — agent di deploy ke prod gantiin software
- Concern: kecepatan agent coding > kecepatan JS framework era

### 7. GPT 5.4 Release & Model Wars

- GPT 5.4 bisa 1M context window
- Amp deep mode langsung upgrade ke Oracle (5.4)
- Codex ga lolos tes 🍓 — lucu tapi Opus aman
- Opus fast mode di Windsurf: 30x multiplier credits 😱

### 8. Qwen 3.5 Local & Chinese Model Ecosystem

- Qwen 3.5 run local di iPhone 17 Pro — mobile AI makin real
- Di Mac mini: ditanya nama aja overthinking 1 menit
- GLM-5-Code akan ada — Chinese model ecosystem berkembang
- Alibaba $3 multimodel deal — cuma ada referral-nya
